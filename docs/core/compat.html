compat.py
¶

Compatibility helpers to bridge the differences between Python 2 and Python 3.

Similar in purpose to six. Not generally intended to be used by third-party software, these are subject to change at any time. Only symbols exported via __all__ are safe to use.

if py3:
	unicode = str
	str = bytes
	items = dict.items
else:
